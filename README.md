# 🚗 Vehicle Market Analysis Dashboard

## 📌 Descripción

Aplicación web desarrollada con Streamlit para analizar el mercado de vehículos usados mediante visualizaciones interactivas.

## 🎯 Objetivo

Identificar los factores que influyen en el precio de los vehículos y detectar patrones relevantes para la toma de decisiones.

## 📊 Dataset

El conjunto de datos incluye:

* Precio
* Año del modelo
* Kilometraje
* Tipo de vehículo
* Condición

## 🧪 Metodología

* Limpieza de datos
* Análisis exploratorio (EDA)
* Visualización de datos
* Desarrollo del dashboard

## 📈 Insights clave

* Existe una relación inversa entre kilometraje y precio
* Los vehículos más recientes presentan menor variabilidad de precios
* Algunos tipos de vehículos mantienen mejor su valor en el mercado

## 🧠 Conclusión

El análisis permite entender la dinámica del mercado de vehículos usados, facilitando decisiones más informadas.

## 🛠️ Tecnologías

* Python (Pandas, Plotly)
* Streamlit

## 🚀 Cómo ejecutar el proyecto

```bash
pip install -r requirements.txt
streamlit run app.py
```
