# Vehicles Dashboard – Sprint 7

Aplicación web desarrollada con Streamlit para el análisis exploratorio de anuncios de venta de vehículos.

## 🚀 Aplicación en producción
La aplicación está desplegada en Render y puede consultarse en el siguiente enlace:

👉 https://vehicles-dashboard-luisfer.onrender.com

## 📂 Contenido del repositorio
- `app.py`: aplicación principal en Streamlit
- `vehicles_us.csv`: dataset de anuncios de vehículos
- `requirements.txt`: dependencias del proyecto
